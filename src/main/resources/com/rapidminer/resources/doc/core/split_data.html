Split Data

Synopsis

Partitions an example set into subsets according to the specified relative sizes.


Description

Divides a data set into the defined partitions and deliver the subsets.


Input

  • example set: expects: ExampleSet

Output

  • partition 1:

Parameters

  • partitions: The partitions that should be created.
  • sampling type: Defines the sampling type of this operator.
  • use local random seed: Indicates if a local random seed should be used.
  • local random seed: Specifies the local random seed
